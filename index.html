Inline Css

In Inline css we write the attributes on the same line

Internal Css

In Internal Css we write the attributes or style inside the html file . The code is written in the head of the html part

External Css

In External Css we write the attributes or style outside the html file . The code is written in the another css file and that css file is linked to the html file using "link tag" and the attribute we use in the link tag are :-
i. "rel" which specifies that what kind of the relation does the css file hold for that html file
ii. "href" which specifies the filename and the extension

CSS Selectors

Welcome to the world of Styling

1. Universal Tag : It select's the entire content in the html tag.
Universal selector uses " * " symbol to style the data
When we add Border attribute in csss file
In Universal Selector , the boarder is applied for each tag present in the html file

2. Element Selector : It select the elements or tags from the html file and styles it accordingly using Css file.
Elementor selector uses elements like h1-h6 , body , p ,...etc
When we add Border attribute in csss file
In Body or Element Selector , the boarder is applied only onces [Outline of the page]

3. Selector List : It selects multiple tags and styles them
Selector list uses " , " symbol to different elements or add element

4. ID Selector : It is the unique identification name given to the particular tag in html File
ID Selector uses " # " symbol to style the data in css file
id="idname" is the attribute to give a unique name to that particular tag.

5. Class Selector : It is the used to select multiple tags and give a name to those tag in html File
Class Selector uses " . " symbol to style the data in css file
class="classname" is the attribute to give a name to those tags.

6. Descandant Selector : It selects the elements that are located somewhere under other elements
It follows the syntax that parent any_child [which means that any parent of that particular child ]
Example :-
Here bold is enclose in h3
here italic is enclosed in bold and h3
here underline is enclose with in italic , bold and h3

7. Direct child Selector : It specifies the parent of that particular child
It uses " > " symbol to specify the direct child selector in css file
It follows the syntax that parent > direct child [which means that direct child is that own child or particular child of that parent]
Example :-
Here bold is enclose in h3
here italic is enclosed in bold and h3
here underline is enclose with in italic , bold and h3

Class Selector uses " . " symbol to style the data in css file
class="classname" is the attribute to give a name to those tags.

Box Model

Specificity: Child selectors are more specific than descendant selectors because they only target direct children.
Scope: Descendant selectors have a broader scope, affecting all matching descendants. Child selectors narrow the scope to direct children only.

Padding : It is used to give the space between the text and the border.
Padding is also known as Internal Spacing
Padding follows clockwise direction.
syntax :- top right bottom left

Margin : It is used to give the space between the border and the screen.
Margin also known as External Spacing
Margin also follows clockwise direction.
syntax :- top right bottom left
